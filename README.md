# 项目特性：
- 📱 **完全响应式**：支持手机、平板、电脑等各种设备，自适应布局
- ⚡ **轻量高效**：纯vue，仅依赖vite
- 🖱️ **交互友好**：点击图片自由缩放，可直接下载，操作简单直观
- 🎭 **丰富表情**：内置多种何意味表情包，涵盖各种场景
- 🔧 **易于扩展**：仅识别文件，不用改动任何其他文件即可轻松添加新的表情包

# 使用方式

## 1.在线访问
访问在线网站: [https://www.何意味.shop](https://www.何意味.shop)

## 2.本地或远程部署
1. 克隆或下载项目文件
```bash
git clone https://github.com/LorienYang/HYW-Meme-Web.git
```

2. 安装[node.js](https://nodejs.org/en/download)

3. 在克隆的目录打开控制台或cmd，输入
```bash
npm install
``` 

4. 运行完成后，输入
```bash
vite
```

5. 本地部署请用浏览器访问 **127.0.0.1:5173** ,远程服务器需要开启反向代理。
